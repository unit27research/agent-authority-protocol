# Mapping: SOC 2

Source: https://www.aicpa-cima.com/resources/landing/system-and-organization-controls-soc-suite-of-services

SOC 2 focuses on controls relevant to security, availability, processing integrity, confidentiality, or privacy.

The Agent Authority Protocol does not provide SOC 2 compliance. It can support control scoping and documentation for agentic systems.

## Security

ARD records:

- tools and systems accessed
- credential type
- action rights
- approval gates
- revocation path

## Availability

ARD records:

- production or critical operations reach
- rollback plan
- incident owner
- kill switch owner
- review cadence

## Processing Integrity

ARD records:

- intended outcomes
- action rights
- approval requirements
- decision logs
- output artifacts

## Confidentiality

ARD records:

- internal data
- sensitive data
- regulated data
- source code access
- retention period

## Privacy

ARD records:

- customer data
- sensitive data
- regulated data
- data access scope
- evidence retention
