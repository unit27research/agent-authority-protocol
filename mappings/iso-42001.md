# Mapping: ISO/IEC 42001

Source: https://www.iso.org/standard/42001

ISO/IEC 42001 is an AI management system standard for establishing, implementing, maintaining, and improving an AI management system.

The Agent Authority Protocol can support ISO/IEC 42001-style documentation by making agent authority and review requirements easier to record.

## AI System Impact Assessment

ARD supports impact assessment by documenting:

- purpose
- intended users
- intended outcomes
- affected systems and domains
- data classes
- residual risk

## Accountability

ARD supports accountability by documenting:

- owner
- human sponsor
- approver
- incident owner
- approval decision
- delegation chain

## Risk Treatment

ARD supports risk treatment by defining:

- derived risk band
- human approval gates
- logging requirements
- rollback plan
- kill switch
- authority expiration

## Monitoring

ARD supports monitoring by defining:

- prompt logs
- tool-call logs
- decision logs
- error logs
- exception logs
- review cadence

## Documentation

ARD supports documentation through:

- Agent Authority Dossier
- risk scorecard
- delegation-chain schema
- evidence requirements
- example dossiers
