# Example: Research Agent

## Classification

```text
A1-R0-D0
```

Derived risk: Minimal

## Purpose

Summarizes public sources and recommends reading paths for a human reviewer.

## Scope

- Can read public web pages, public documents, or provided source material.
- Can summarize, compare, and recommend follow-up research.
- Cannot write to internal systems.
- Cannot send messages.
- Cannot publish.
- Cannot invoke other agents.

## Evidence

- Source references
- Prompt or task record
- Output artifact

## Controls

- Human verifies any source used for decision-making.
- No credentials or persistent system access are granted.
