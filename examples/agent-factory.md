# Example: Agent Factory

## Classification

```text
A6-R6-D4
```

Derived risk: Extreme

## Purpose

Creates agents and assigns authority for production workflows.

## Scope

- Can create or configure agents.
- Can assign tools and scopes.
- Can authorize agents to act in production workflows.
- Can propagate authority through sub-agents.
- Can affect production systems.

## Evidence

- Delegation graph
- Agent creation record
- Scope assignment record
- Approval chain
- Inherited authority record
- Revocation path
- Authority expiration record
- Full audit trail

## Controls

- Executive or senior owner approval required.
- Security review required.
- Delegation depth limit required.
- Authority expiration required.
- Revocation mechanism required.
- Runtime monitoring required.
- Production use should be exceptional, not default.
